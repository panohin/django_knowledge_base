
Модель - класс django.db.models.Model.

Каждый аттрибут модели представляет поле таблицы.

Необязательные, но популярные аттрибуты полей:
1. null
Если True - Django установит NULL в базе данных в случае отсутствия данных. По дефолту False.
2. blank
null для базы данных, а blank для формы. Т.е., если blank=True - валидатор формы позволит полю быть пустым.
3. choices
Список из кортежей, содержащих по два стринга. Первый из которых сохраняется в базе данных, а второй отображается в форме.
4. default
5. help_text
Дополнительная информация. Отображается в форме.
6. primary_key
По дефолту primary_key=False.
Джанго сам создаёт IntegerField id и присваевает ему PK.
7. unique

Primary key
Если указать аттрибут primary_key=True, Джанго, увидев это, не создаст автоматически столбец id.

verbose_name (пер. подробное имя)
Каждый тип поля имеет первый позиционный аргумент verbose_name.
Если он не указан, verbose_name будет автоматически создан из имени поля, заменив нижние подчеркивания пробелами.
